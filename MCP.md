# MCP 配置建议

下面是建议接入的 MCP 方向，便于后续 Excel/Word/CAD 处理功能开发与调试。

## 推荐 MCP 能力
- `filesystem`：读取大文件、批量遍历目录、统一路径策略
- `sqlite`：本地缓存与结果索引（与当前缓存机制一致）
- `office`：Excel/Word 读写与批量处理
- `ocr`：图片或扫描版文档文字识别
- `diff`：文档对比与相似度分析

## 说明
- 本项目优先离线运行，MCP 应支持本地部署或离线模式。
- 如暂时没有 MCP 服务，可保留配置占位，后续按环境补齐。
- GitHub 自动同步建议使用 GitHub Actions（CI）而非本地自动 push，避免凭据与误提交风险。
