# AGENTS.md

本项目后续会扩展 Excel/Word/CAD 批量处理功能。为了提高开发效率与准确度，建议在协作时按如下 **Agent 角色分工** 与 **工作约定** 执行。

## Agent 角色建议
- `planner`：拆解需求，产出实现方案、数据流、测试用例清单。
- `implementer`：按方案落地代码、配置、打包脚本。
- `reviewer`：检查性能、并发、兼容、错误处理、回退策略。

## 项目协作约定
- 所有新增功能必须支持 **离线使用**，避免强依赖外部服务。
- 所有批量任务应支持 **取消** 与 **进度显示**。
- 任何耗时任务优先 **后台线程**，避免 UI 卡顿。
- 结果必须支持 **导出 CSV/XLSX**。
- 日志必须落盘，可定位问题。
- 默认不在本地自动执行 `git push`。如需自动同步到 GitHub，请使用 CI（GitHub Actions）或显式脚本触发。

## 功能扩展优先级建议
- Excel 合并/筛选/导出
- Word 内容提取与差异对比
- DWG 文字批量替换
- Word/Excel 文字批量替换
